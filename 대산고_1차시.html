01

VSCode란?

가볍고 강력한 코드 에디터

💻 Visual Studio Code

  • Microsoft가 만든 무료 코드 편집기
  • Windows / Mac / Linux 모두 지원
  • 확장 프로그램으로 기능 무한 확장
  • 전 세계 개발자 70%+ 사용

📥 설치 방법

  • code.visualstudio.com 접속
  • OS에 맞는 버전 다운로드
  • 설치 후 한국어 팩 설치

✨ 왜 VSCode?

  • AI 도구와의 통합이 가장 잘 됨
  • 터미널 내장 — 코드+실행 한 곳에서
  • 거의 모든 프로그래밍 언어 지원

AI에게 잘 물어보는 법

프롬프트가 좋으면 결과도 좋다

❌ 이렇게 하면 안 됨

  • "코드 짜줘"
  • "에러 고쳐줘"
  • "이거 왜 안 돼?"

→ 맥락 없이 던지면 엉뚱한 답

✅ 이렇게 하면 좋음

  • "Python으로 1~100 소수 구하는 함수"
  • "이 코드에서 TypeError 원인 설명해줘"
  • "CNN 모델에 Dropout 추가해줘" (Claude Code)

→ 구체적 맥락 + 원하는 결과를 명시

💡 프롬프트 공식

[언어/도구] + [구체적 작업] + [조건/맥락]

"Python으로 MNIST 데이터셋 불러와서 CNN 모델 학습해줘. Epoch 10, batch 32" (Codex)

01

좋은 탐구 주제란?

탐구는 "소개"가 아닙니다
질문 → 조사 → 비교/분석 → 정리 과정이 있어야 탐구입니다

✅ 좋은 주제 체크리스트

  • 한 문장으로 질문이 되는가?
  • 내가 궁금한 점이 분명한가?
  • 직접 조사/실험/비교가 가능한가?
  • 자료를 찾을 수 있는가?
  • 정해진 기간 안에 끝낼 수 있는가?
  • 결과를 표/그래프로 정리할 수 있는가?

03

자료 조사 방법

검색어 만드는 법부터 시작

🔍 검색 팁

  • 키워드 2~3개 조합하기
  • "정의 / 원리 / 비교 / 사례" 확장
  • 한 자료만 보지 말고 교차 확인
  • 영어 검색도 시도해보기

📝 검색어 예시

CNN MNIST accuracy comparison
hybrid retrieval RAG comparison
MITM attack encryption principle
load balancing round robin 비교

✅ 신뢰할 만한 자료

  • 대학/연구기관 자료
  • 공식 문서 · 논문
  • 전문 기관 기술 설명

⚠️ 주의할 자료

  • 출처 불분명한 블로그
  • 광고성 · 근거 없는 글
  • 복붙 느낌의 AI 생성 글

2차시까지 할 과제

주제는 오늘 확정 → 2차시에는 조사 결과를 점검합니다

과제 1 · 핵심 개념 정리

  • 주제에 필요한 용어/개념 3개
  • 각각 3~5줄로 설명
  • 출처 함께 정리
  • 예: RAG란? 하이브리드 검색이란?

과제 2 · 참고 자료 3개

  • 제목 / 출처 (링크)
  • 핵심 내용 2~3줄 요약
  • 왜 이 자료가 믿을 만한지 한 줄
  • 블로그 1개만 보고 끝내지 않기!

과제 3 · 탐구 계획서

  • 무엇을 비교/분석할 것인지
  • 어떤 도구를 쓸 것인지
  • 예상 결과 (가설)
  • 3~4차시 일정 계획

🎬 추가 과제: 2차시에 10분 이내 발표 — 탐구 계획 + 자료조사 결과를 정리해서 발표하기

📅 2차시 전까지 제출 · 형식 자유 (문서/PPT/노션 등) · 발표 자료는 화면 공유 준비
