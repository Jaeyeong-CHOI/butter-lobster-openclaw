# Notes Prompt Template

Use this structure for the final notes object:

- title
- one_line
- summary
- key_points (3-8)
- terms (important entities/concepts)
- action_items (practical next steps)
- confidence_notes (OCR ambiguity warnings)

Guidelines:

- Preserve numbers, equations, and named entities when visible.
- Mark uncertain text explicitly; do not fabricate.
- Keep summary concise but information-dense.
