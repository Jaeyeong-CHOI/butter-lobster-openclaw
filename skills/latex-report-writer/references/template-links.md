# Report template links

Canonical templates should be provided by the repository owner via private storage.
Do not store personal or organization-private links in public repositories.

## Required inputs (fill privately)

- Latest final report PDF: `<PRIVATE_LINK_REPORT_PDF>`
- Reusable LaTeX template `.tex`: `<PRIVATE_LINK_TEMPLATE_TEX>`
- Template preview PDF: `<PRIVATE_LINK_TEMPLATE_PREVIEW>`

## Usage note

If private links are unavailable, use local templates in `assets/templates/` as fallback.
