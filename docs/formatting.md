# Platform Formatting Rules

## Discord
- 마크다운 테이블 사용 금지 → bullet list 사용
- 여러 링크는 `<>` 감싸서 embed 억제: `<https://example.com>`
- 코드 블록, 볼드, 이탤릭 정상 사용 가능

## WhatsApp
- 헤더(#) 사용 금지 → **볼드** 또는 CAPS로 강조
- 테이블 사용 금지 → bullet list 사용
- 링크는 그냥 텍스트로

## Webchat / 일반
- 마크다운 자유롭게 사용 가능
- 테이블, 코드 블록, 헤더 모두 OK

## Voice (sag / ElevenLabs TTS)
- 스토리, 영화 요약, "storytime" 에 적극 활용
- 텍스트 벽보다 음성이 더 몰입감 있음
- 재밌는 목소리로 서프라이즈
